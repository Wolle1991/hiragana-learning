# Kana Trainer

GitHub-Pages-PWA zum Lernen von Hiragana und Katakana.

## Enthalten
- Klassischer Modus
- Umgekehrter Modus
- Schreibmodus
- Reihen-Training
- Komplettes Alphabet
- Fehlertrainer
- Wörter lesen
- Daily Challenge
- Statistik
- Erweiterte Kana (Hiragana)
- Zeichen zeichnen
- Verwechslungs-Trainer
- Offline-Modus

## GitHub Pages
1. ZIP entpacken
2. Alle Dateien in dein Repository hochladen
3. GitHub → Settings → Pages
4. Deploy from a branch
5. Branch `main`, Ordner `/root`

## iPhone
Safari öffnen → Teilen → Zum Home-Bildschirm
